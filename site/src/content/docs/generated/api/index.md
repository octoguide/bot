The [octoguide](https://www.npmjs.com/package/octoguide) package
to use the API is the same as the [standalone CLI](https://octo.guide/cli)
and its code is directly used by the [GitHub Action](https://octo.guide/get-started#installation).

For installation instruction see
[Installing the OctoGuide Package](https://octo.guide/docs/installing-octoguide-package).

## Functions

| Function | Description |
| ------ | ------ |
| [cliReporter](/generated/api/functions/clireporter/) | Formats a rule report as used by the standalone CLI. |
| [markdownReporter](/generated/api/functions/markdownreporter/) | Formats a rule report as used by the GitHub Actions Workflow. |
| [runOctoGuideRules](/generated/api/functions/runoctoguiderules/) | Runs OctoGuide's rules to generate a list of reports for a GitHub entity. The entity can be provided as either a URL string (which will be fetched from the GitHub API) or pre-existing entity data. |

## Interfaces

| Interface | Description |
| ------ | ------ |
| [RunOctoGuideRulesOptions](/generated/api/interfaces/runoctoguiderulesoptions/) | Settings for running [runOctoGuideRules](/generated/api/functions/runoctoguiderules/). Only `entity` is required. |
| [RunOctoGuideRulesResult](/generated/api/interfaces/runoctoguiderulesresult/) | Returned data from running [runOctoGuideRules](/generated/api/functions/runoctoguiderules/). |
| [CommentEntity](/generated/api/interfaces/commententity/) | A resolved comment entity from GitHub. Can be a comment on an issue, pull request, or discussion. |
| [DiscussionEntity](/generated/api/interfaces/discussionentity/) | A resolved discussion entity from GitHub. |
| [IssueEntity](/generated/api/interfaces/issueentity/) | A resolved issue entity from GitHub. |
| [PullRequestEntity](/generated/api/interfaces/pullrequestentity/) | A resolved pull request entity from GitHub. |
| [RuleReport](/generated/api/interfaces/rulereport/) | Violation report generated by a rule for a GitHub entity. |
| [RuleReportData](/generated/api/interfaces/rulereportdata/) | Report data for a specific [RuleReport](/generated/api/interfaces/rulereport/) violation. |
| [Rule](/generated/api/interfaces/rule/) | Defines how to analyze entities for a single best practice. |
| [RuleAbout](/generated/api/interfaces/ruleabout/) | Metadata about a rule, as used to define the rule. |
| [RuleContext](/generated/api/interfaces/rulecontext/) | Shared context provided to rules when they run on an entity. |

## Type Aliases

| Type Alias | Description |
| ------ | ------ |
| [Entity](/generated/api/type-aliases/entity/) | A resolved entity retrieved from the GitHub API. Can be a comment, discussion, issue, or pull request. |
| [RuleListener](/generated/api/type-aliases/rulelistener/) | Rule property called if the rule is run on the corresponding entity type. |
| [RuleReporter](/generated/api/type-aliases/rulereporter/) | Context function for a rule to register a new violation. |
